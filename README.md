# ProxyEnvSwitcher
「ProxyEnvSwitcher」 is a lightweight tool for proxy management on Windows (Mac support planned). It offers one-click switching, automated profiles, and custom rules, ideal for developers. No command-line needed—its intuitive UI simplifies setup. 「ProxyEnvSwitcher」 是一款轻量级代理管理工具，现支持 Windows，计划支持 Mac。一键切换、自动配置、自定义规则，适用于开发场景。无需命令行，直观界面降低门槛。
